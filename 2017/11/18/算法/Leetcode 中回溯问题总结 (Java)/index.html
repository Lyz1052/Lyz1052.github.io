A general approach to backtracking questions in Java (Subsets, Permutations, Combination Sum, Palindrome Partitioning)

  • Subsets

    • 子数组问题，相当于求集合的所有排列组合
    • 不需要排序

  • Subsets II

    • 包含重复元素的子数组问题
    • 需要排序
    • 通过判断元素与之前一个元素是否相同，可以保证重复元素仅仅只能以同一种顺序加入序列中，保证该序列仅记录一次

  • Permutations

    • 求全排列，记录递归深度，通过一个标准的回溯流程实现
    • 不需要排序

  • Permutations II

    • 包含重复元素的全排列问题
    • 需要排序
    • 需要记录元素是否被使用过（不能仅仅根据判断临时序列中是否包含当前元素，来判断是否将当前元素加入临时序列）
    • 对于重复（判断规则同Subsets II）且并未使用过的元素，通过判断其前一个重复的元素是否使用过（如果未使用过，则跳过当前元素），可以保证包含重复元素的序列仅添加一次

  • Combination Sum

    • Subsets II中的回溯添加一个剩余和的条件
    • 可以添加重复元素

  • Combination Sum II

    • 类似Subsets II，先排序，再通过与前一个元素比较进行判重
    • 添加剩余和条件
    • 不可以添加重复元素

  • Palindrome Partitioning

    • 通过判断当前字符串是否是镜像字符串，来判断是否进行下一步回溯
  • 发布日期: 2017-11-18
