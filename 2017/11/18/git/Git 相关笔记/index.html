git

http://www.jianshu.com/p/fe76f2890a14

特性

  • 不记录文件变化，只记录文件快照，不同版本中，未做过改动的文件的索引指向相同的文件快照
  • 大多数操作仅仅添加数据，定期推送到其他Git仓库可以防止数据丢失

文件的状态

  • Git目录下的文件状态

    • modified
    • commited
    • staged

  • 工作目录下的文件状态

    • untracted
    • unmodified

文件流转的工作区域

  • working directory(dir) 工作目录，git资源所在的文件夹
  • staging area 暂存区域
  • git directory(repo) git目录，git资源库

.git（Git目录）

  • 用于保存元数据和对象数据库
  • 使用

$ git config –global user.name “John Doe”
$ git config –global user.email johndoe@example.com

git help
git –help
```

git常用的操作流程

  • 见note

branch（分支）

  • git branch列出当前所有分支的清单，带*号的是当前分支
    • -v 列出分支版本号
    • --merged列出当前分支的上游分支
